transition 어떤 상태(state)에서 다른 상태(state)로의 "변화"를 애니메이션으로 만드는 방법 transition은 상태가 없는 요소에 적어야한다. transition: all (변화하는 모든것들을 선택) 바뀌기전 요소(전 상태)에 trainsition은 바꿀 결과(새 상태)를 적었을 때 적용된다. ex) transition: background-color: wheat; 10s ease-in-out a:hover { background-color: tomato; color: wheat; } 로 이미 바꾼 결과를 적어주었기 때문에 적용됨.
